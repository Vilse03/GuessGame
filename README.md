Gissningslek / Guess the Number inspirerat program

Min idé är att skapa ett spel där du ska gissa rätt på ett tal. 
I mitt spel tänker jag att det ska finnas olika svårighetsgrader och möjlighet att begränsa dina gissningar. 
Programmet kommer att spara resultat från spelet i en fil.

Körning:
Körs i terminalen.
Programmet kommer att starta, hälsar på dig etc. 
Därefter kommer programmet ge dig alternativ, svårighetsgrad/antal försök.
När du har gissat rätt/eller fel så kommer spelet avslutas och ge dig information ifall du vann eller inte.


Innehål 
Inmatningsvalidering - heltal, inga decimaler. Checkar efter ogiltiga inputs
Data - sparas (namn, svårighetsgrad, max-tal, hemligt tal, försök, vinst/förlust)
Minst två tester som kontrollerar logiken
Klass '' med minst två metoder som hanterar spelet.
